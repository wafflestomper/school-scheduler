CSV Format:

Your CSV file should include the following columns:

  • username (required) - Unique username for the user
  • user_id (required) - Unique identifier for all users
  • email (optional) - Email address
  • first_name (optional) - First name
  • last_name (optional) - Last name
  • role (optional, defaults to 'student') - One of: student, teacher, admin
  • grade (optional) - Grade level (for students)
  • gender (optional) - M or F

Example:

username,user_id,email,first_name,last_name,role,grade,gender
student1,S12345,student1@school.edu,John,Doe,student,7,M
teacher1,T789,teacher1@school.edu,Jane,Smith,teacher,,F

Note: Common formats for User IDs:

  • Students: S12345 (S followed by numbers)
  • Teachers: T789 (T followed by numbers)
  • Admins: A123 (A followed by numbers)
